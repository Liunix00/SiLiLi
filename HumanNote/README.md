# HumanNote

`HumanNote/` 是**人类权威数据源**（source of truth）：项目的 Plan / Progress / Idea，以及每天的 Steps 记录都写在这里。

默认情况下，本仓库的 `.gitignore` 会忽略 `HumanNote/`（避免把个人工作记录、频繁变动内容或可能的敏感信息提交到 Git）。如果你希望把部分内容纳入版本管理，请调整 `.gitignore` 的白名单规则。

## 目录约定

- `Steps/`：按“年 / 人名 / 日期”组织的每日工作记录
- `Projects/`：按项目目录组织的计划与进展（以及 `history/` 归档历史）
