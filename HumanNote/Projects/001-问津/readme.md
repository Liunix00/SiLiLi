# 001-问津
问津：询问渡口在哪里，就好像患者和陪诊师来询问应该去哪里就医

## 基本信息
- status: 🟡 进行中
- 创建时间: 2026-01-31
- 负责人: 刘玮康

## 项目目标
构建数据库，和相应的查询思路，为回答患者和陪诊师关于就医的各种问题，提供事实依据

## 项目背景
大语言模型的通用能力无法覆盖就医细节，网络搜索无法保证资料真实性。就医过程中需要的各种信息，分散在网络和线下的各种地方，相应数据的收集和利用有巨大的价值。

## 核心思路
1. 创造 【内部自建立】医院医生科室数据库 就医流程数据库 医保信息数据库 药品信息数据库 疾病信息数据库 检查治疗数据库 以及
【允许外部录入】公开信息数据库 陪诊师录入数据库
2. 完成内部数据库建立
3. 完成外部数据库的录入机制
4. 建立数据库内容应用的机制
5. 建立内外部数据库的通信机制
