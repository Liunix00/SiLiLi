# 002-灵枢
灵代表AI，枢代表公司的各个部门。灵枢计划是希望各个部门，从人到工作模式，可以与最新的Agent发展接轨

## 基本信息
- status: 🟡 进行中
- 创建时间: 2026-03-21
- 负责人: 刘玮康

## 项目目标

 顺应AI智能体高速发展趋势，由算法组牵头，使用智能体技术，为各部门进行赋能，提高其工作效率和工作质量；同时带动各部门，提高对智能体的认知和使用能力

## 项目背景
 agent应用快速发展，企业级智能体平台，openclaw，claude code，个人级别智能体应用等井喷。agent化已成必不可挡趋势

## 核心思路

 1. 已经收集各部门需求，将其整理为三个模块，做进App中，算法和技术组开发，算法组提供AI使用建议这三类。第一类已经加入别的project，第三类只需要算法组提供一个AI实践的文档给全体员工，第二类需要重点关注追踪
 2. 目前已经提取需求11个，放入 **002-灵枢-plan.md** 中，设计智能体浏览器操作，信息获取
